사용자 가이드

1. 기초 외환 입력

기초 외환 수량 및 환율을 입력합니다. 저장이 된 경우에는 불러오기를 이용합니다.

2. 환율 데이터

환율 엑셀 파일을 업로드 하면 자동으로 저장이 됩니다. 다른 PC에서 사용시에는 새로 업로드 해야 합니다.

3. 은행 거래 업로드

은행 엑셀 파일을 업로드 합니다. 파일명은 상관이 없습니다.

4. 저장 기록 삭제

이전 저장 기록 목록에서 항목을 선택 후 삭제 버튼을 누르면 해당 스냅샷이 영구 삭제됩니다.

⚠️ 오류 자주 발생! — 제목열 중복 키워드 주의

거래 내역 엑셀의 제목 행에 입금·출금 관련 단어가 2개 이상 있으면 오류가 발생합니다.

❌ 문제가 되는 예시

"입금액" + "입금의뢰인명" → 입금 키워드 중복!

"출금액" + "출금거래처" → 출금 키워드 중복!

✅ 해결 방법

"입금의뢰인명" → "의뢰인명" 으로 변경

"출금거래처" → "거래처" 로 변경
